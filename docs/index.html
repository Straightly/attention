Manage your todos and sync to GitHub

🔐 GitHub Authentication

Enter your GitHub Personal Access Token to get started. Your token will be stored securely in your browser.

How to create a token:

  1. Go to GitHub Settings → Personal Access Tokens
  2. Click "Generate new token" (Fine-grained)
  3. Token name: Attention Todo App
  4. Repository access: Only select repositories → Straightly/attention
  5. Permissions: Repository permissions → Contents → Read and write
  6. Click "Generate token" and copy it
📄 Managing: ToDos/List.json
