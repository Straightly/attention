📝 Attention - Todo Manager

Manage your todos and sync to GitHub

⚙️ Setup Required

Please configure your GitHub token in config.js

  1. Go to GitHub Settings → Personal Access Tokens
  2. Click "Generate new token" (Fine-grained)
  3. Repository access: Select only Straightly/attention
  4. Permissions: Contents → Read and write
  5. Copy the token and paste it in config.js
